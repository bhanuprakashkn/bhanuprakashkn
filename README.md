## Flagship Projects

### 🧠 Stroke Clinical Decision Support System (CAD)
Designed and deployed an end-to-end stroke CAD system covering **ischemic, hemorrhagic, and probabilistic stroke** assessment for emergency room use. Deployed across Singapore health clusters. *Asian Innovation Award finalist · RSNA Certificate of Merits.*

### 🧬 SG Normative Brain Template & Population-Based Volumetry Atlas
Deep learning framework to build a **Singapore-specific ageing brain reference template** — the first population-normalised volumetry atlas for an Asian cohort. Enables personalised neurological assessment at scale. *(NHIC Innovation Grant)*

### 🏃 AI-Based Sarcopenic Obesity & Metabolic Profiling Framework
Clinical AI pipeline for phenotyping sarcopenic obesity in a geriatric population using multimodal data. Designed to integrate directly with clinical assessment workflows in Singapore's ageing-in-place programmes.

### 🫀 Multi-site Cardiac MRI Analysis App
Built a cardiac analysis application capable of handling diverse MRI cardiac studies across multiple clinical sites — standardising acquisition and analysis workflows for research and clinical trials.

### 🔬 Schizophrenia Subtype Classification via Neuroimaging Biomarkers
Designed a machine learning pipeline for classification of schizophrenia subtypes using neuroimaging — enabling personalised treatment pathways. Applied unsupervised clustering to identify biologically meaningful patient subgroups.

### 📐 Brain Atlas Landmark Registration
Developed novel anatomical landmarks for improved brain atlas registration accuracy, adopted in multi-centre neuroimaging studies. *Multiple granted patents.*
